# Quick Start Guide

### Prerequisites
- .NET 9 SDK installed

### Steps

1. **Open Terminal** in the `StoryBoardBud/` folder

2. **Run the app:**
   ```bash
   dotnet run
   ```

3. **Open your browser** to: `https://localhost:5294`

4. **Test Accounts** (already created):
   - **Admin**: `admin` / `AdminPassword123!`
   - **Test User**: `testuser` / `TestPassword123!`
